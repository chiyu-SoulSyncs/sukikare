# Schedule Assistant — デザイン計画

## アプリコンセプト
Googleカレンダーと連携し、指定した日付・週の空き時間を自動抽出して、日程調整メッセージを生成・コピーできるビジネス向けモバイルアプリ。

## 画面一覧

### 1. ログイン画面 (`/login`)
- Googleアカウントでサインインボタン
- アプリロゴ・説明文

### 2. ホーム画面 (`/(tabs)/index`) — 日程検索
- 検索モード切替：「日付指定」「週指定」タブ
- **日付指定モード**: カレンダーUIで複数日付を選択（タップでトグル）
- **週指定モード**: 週セレクター（前週/次週ナビゲーション）
- 検索設定パネル（折りたたみ式）:
  - 時間帯フィルター（営業時間 9:00-18:00 / カスタム / 時間外OK）
  - 最小空き時間（30分・1時間・1.5時間・2時間）
  - 候補件数（3件・5件・無制限）
  - 所要時間（30分・1時間・1.5時間・2時間）
- 「空き時間を検索」ボタン（大きめ・目立つ）

### 3. 結果・メッセージ生成画面 (`/result`)
- 抽出された空き時間スロット一覧（チェックボックスで選択）
- メッセージ設定:
  - 宛先（任意入力）
  - 件名（任意入力）
  - 敬語レベル（ビジネス丁寧語 / カジュアル / タメ口）
  - フォーマット選択（箇条書き / 表形式 / 文章形式）
- 「メッセージを生成」ボタン
- 生成されたメッセージプレビュー
- コピーボタン（大きめ・ハプティクス付き）
- シェアボタン（システムシェアシート）

### 4. プリセット画面 (`/(tabs)/presets`)
- 保存済みプリセット一覧（カード形式）
- プリセット内容: 時間帯・最小空き時間・候補件数・敬語レベル・フォーマット
- 新規プリセット追加ボタン
- プリセット削除（スワイプ or 長押し）
- プリセットをホームに適用

### 5. 設定画面 (`/(tabs)/settings`)
- Googleアカウント情報・ログアウト
- 複数カレンダー選択（チェックリスト）
- 営業時間のデフォルト設定
- ダークモード切替
- アプリバージョン情報

## キーユーザーフロー

### フロー1: 日程候補メッセージ作成
1. ホーム画面で日付を選択（複数タップ）
2. 検索設定を調整（時間帯・最小時間・候補数）
3. 「空き時間を検索」タップ → Googleカレンダー API 呼び出し
4. 結果画面で空き時間スロットを選択
5. 宛先・件名・敬語レベル・フォーマットを設定
6. 「メッセージを生成」タップ
7. プレビューを確認 → コピー or シェア

### フロー2: プリセット利用
1. プリセット画面から保存済みプリセットをタップ
2. ホーム画面に設定が自動反映
3. 日付を選択して検索

## コンポーネント設計

- `DatePicker`: 月カレンダーUI、複数日タップ選択
- `WeekSelector`: 週ナビゲーション、曜日ごとのトグル
- `SearchSettingsPanel`: 折りたたみ式設定パネル
- `TimeSlotCard`: 空き時間スロット（選択可能カード）
- `MessagePreview`: 生成メッセージ表示エリア
- `PresetCard`: プリセット表示カード
- `FormatSelector`: フォーマット選択ボタングループ
